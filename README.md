# Scoreboard

Scoreboard is a simple game score tracker that lets you record and view scores during a game session. I built this project to practice working with vanilla JavaScript, HTML, and CSS while making a useful little utility for tracking points in games.

## Features

- Track scores for two players or teams
- Buttons to increase each score
- Simple user interface
- Works in the browser

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- GitHub  

## Getting Started

To run this project on your local machine, follow these steps.

### Prerequisites

- A web browser
- No backend or database needed

### Installation

1. Clone this repository
2. Navigate into the project folder
3. Open `index.html` in your browser

## Usage

- Use the increase buttons to add points to each score
- Refresh the page to reset scores
- This is good for casual games and practice
